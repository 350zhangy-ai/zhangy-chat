## 发布说明

### 🎉 核心升级

Zhangy Chat R4 实现从"数据匹配式问答"到"智能逻辑推理"的突破，从"程序化安慰"到"真实情感共情"的升级。

### ✨ R4 新增功能

#### 🧠 逻辑推理引擎
- **语义解析**: 识别 5 种问题类型（为什么/怎么做/要不要/哪个更好/一般性）
- **逻辑推理**: 基于因果、流程、决策、比较等逻辑推导答案
- **情境适配**: 结合用户情绪调整回答角度和表述方式
- **答案生成**: 输出有逻辑、有温度的回答，而非标准答案

**支持的问题类型**:
| 类型 | 疑问词 | 推理方式 |
|------|--------|----------|
| 因果推理 | 为什么/为啥 | 分析原因和结果 |
| 流程推理 | 怎么/如何 | 给出行动框架 |
| 决策推理 | 要不要/该不该 | 梳理决策思路 |
| 比较推理 | 哪个更好/区别 | 对比分析维度 |

#### 💚 情感计算引擎
- **深度情感识别**: 8 种情绪（焦虑/沮丧/生气/疲惫/压力/开心/自豪/孤独）
- **情感记忆**: 记住用户近期情绪（7 天自动清除），实现上下文情感衔接
- **真实共情回复**: 先接纳→再理解→最后轻量建议，不是程序化安慰
- **情感强度调节**: 弱/中/强三档，用户可自定义共情深度

**共情策略**:
- **负面情绪**: 换谁都会难过→我知道你很努力→先休息一下
- **正面情绪**: 太为你开心了→这是你应得的→好好庆祝一下

#### 🔗 智能情境适配
- 负面情绪时：先共情，再解答
- 正面情绪时：先共鸣，再强化
- 中性问题时：直接解答，适度关怀

### 📝 新增 CMD 指令

```
# 逻辑推理模式
/logic <问题>
示例：/logic 为什么坚持很难
      /logic 为什么加班越久效率越低

# 情感管理
/emotion check        - 查看情感状态
/emotion clear        - 清除情感记忆
/emotion level [弱/中/强] - 设置情感强度
```

## 简介

zhangy-chat R3 是一个高效、专业的本地 AI 助手，采用豆包风格的现代化 UI 设计，提供智能问答、任务管理、目标规划、习惯打卡等功能。所有数据本地运行，隐私安全。

## 核心特性

### 🎨 豆包风格 UI
- **现代简洁**: 采用豆包同款配色方案，视觉舒适
- **气泡对话**: 用户/AI 消息采用不同风格气泡，清晰易读
- **心情标签**: 5 种心情选择，调整 AI 回应风格
- **场景预设**: 4 类预设模式，一键切换

### 🤖 智能问答
- **超全知识库**: 覆盖饮食、编程、工作、学习、情感、健康、购物、旅行等
- **像人一样思考**: 通用问题智能分析，给出有温度的建议
- **情感识别**: 识别用户情绪，提供共情回应
- **危机干预**: 自杀倾向检测，提供心理援助热线

### 📋 任务管理
- **待办事项**: 添加、删除、标记任务，支持优先级
- **目标规划**: 设定目标，拆解里程碑，进度追踪
- **习惯打卡**: 每日/每周习惯，连续打卡统计
- **复盘报告**: 生成周期性复盘，查看成长轨迹

### 💾 数据管理
- **本地存储**: 所有数据存在本地，不上传云端
- **备份恢复**: 一键备份，随时恢复
- **导出功能**: 支持导出 TXT、Excel 格式

## 快速开始

### 环境要求

- Python 3.8+
- 8GB+ RAM
- Windows 10+ / macOS 12+ / Linux

### 安装

```bash
cd zhangy_chat
pip install -r requirements.txt
```

### 运行

```bash
# GUI 图形界面模式（推荐）
python gui.py

# CMD 命令行模式
python main.py -cmd
```

## 使用指南

### GUI 界面

1. **对话页面**: 输入问题，获取 AI 建议
2. **任务页面**: 管理待办事项
3. **目标页面**: 规划长期目标
4. **习惯页面**: 打卡日常习惯
5. **设置页面**: 配置内存、心情、预设

### CMD 指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/add <标题>` | 添加任务 | `/add 学习 Python` |
| `/del <ID>` | 删除任务 | `/del abc12345` |
| `/list [状态]` | 查看任务 | `/list pending` |
| `/mark <ID> <状态>` | 标记任务 | `/mark abc12345 completed` |
| `/goal add <标题>` | 添加目标 | `/goal add 考研上岸` |
| `/goal list` | 查看目标 | `/goal list` |
| `/habit add <名称>` | 添加习惯 | `/habit add 早起` |
| `/habit check <ID>` | 习惯打卡 | `/habit check abc12345` |
| `/backup` | 备份数据 | `/backup` |
| `/export <格式>` | 导出数据 | `/export excel` |
| `/mem [8/16/32/64]` | 设置内存 | `/mem 16` |
| `/mood [心情]` | 设置心情 | `/mood 高效` |
| `/preset [预设]` | 切换预设 | `/preset 办公` |
| `/status` | 查看状态 | `/status` |
| `/gui` | 切换图形界面 | `/gui` |
| `/exit` | 退出程序 | `/exit` |

## 配置说明

### 内存配置

| 内存 | 缓存 | 并发 | 数据加载 |
|------|------|------|----------|
| 8GiB | 64MB | 2 | 分块加载 |
| 16GiB | 256MB | 4 | 分块加载 |
| 32GiB | 512MB | 8 | 混合加载 |
| 64GiB | 1024MB | 16 | 全量加载 |

### 数据存储

- **任务数据**: `data/tasks.json`
- **目标数据**: `data/goals.json`
- **习惯数据**: `data/habits.json`
- **备份目录**: `backups/`

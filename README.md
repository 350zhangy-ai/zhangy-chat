## 简介

Zhangy Chat R3 是一个智能适配的本地 AI 助手，在 R2 基础上新增**内存配置**、**心情选择**、**场景预设**三大核心功能。它可根据用户硬件环境、情绪状态与使用场景动态调整响应策略，提供真正的个性化陪伴体验。

## 核心特性

### 🎯 R3 新增功能

#### 内存配置 (8/16/32/64GiB)
- **智能适配**: 自动检测物理内存，推荐最优配置
- **性能优化**: 根据内存配置调整缓存大小、并发数、数据加载方式
- **低配保护**: 8GiB 模式下内存占用≤150MB，防止溢出
- **高配加速**: 64GiB 模式下全量加载，响应更快

#### 心情选择 (5 种标签)
- **😟 焦虑/低落**: 共情治愈，温柔安抚
- **🎯 高效/专注**: 简洁干练，直奔主题
- **😊 轻松/愉悦**: 亲切活泼，增加互动
- **🌿 平静/温和**: 温和平稳，给予安全感
- **😴 疲惫/需要休息**: 关怀为主，强调休息

#### 场景预设 (4 类模式)
- **💼 高效办公**: 任务拆解、时间管理、简洁回应
- **📚 备考冲刺**: 学习计划、知识点梳理、打卡提醒
- **🍵 休闲陪伴**: 轻松聊天、生活建议、兴趣话题
- **💚 情绪疏导**: 共情倾听、压力缓解、心理调节

### 🔄 R2 保留功能

| 功能类别 | 说明 |
|----------|------|
| **双界面切换** | GUI 图形界面 + CMD 命令行，无缝切换 |
| **任务管理** | 待办 CRUD、优先级排序、拖延提醒 |
| **目标规划** | 目标拆解、里程碑、进度追踪、复盘报告 |
| **习惯打卡** | 每日/每周习惯、连续打卡统计 |
| **数据管理** | 本地存储、备份恢复、导出 TXT/Excel/CSV |
| **AI 助手** | 智能问答、情绪疏导、内容辅助 |

## 快速开始

### 环境要求

- Python 3.8+
- 8GB+ RAM
- Windows 10+ / macOS 12+ / Linux (Ubuntu 20.04+)

### 安装

```bash
cd zhangy_chat
pip install -r requirements.txt
```

### 运行

```bash
# GUI 图形界面模式
python gui.py

# CMD 命令行模式
python main.py -cmd

# 或统一入口（默认 GUI）
python main.py
```

## CMD 指令列表

### R3 新增指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/mem [8/16/32/64]` | 设置内存配置 | `/mem 16` |

## 配置说明

### 内存配置参数

| 内存 | 缓存大小 | 最大并发 | 批次大小 | 数据加载 |
|------|----------|----------|----------|----------|
| 8GiB | 64MB | 2 | 16 | chunked |
| 16GiB | 256MB | 4 | 32 | chunked |
| 32GiB | 512MB | 8 | 64 | mixed |
| 64GiB | 1024MB | 16 | 128 | full |

│   ├── __init__.py         # 模块导出 (R3 更新)

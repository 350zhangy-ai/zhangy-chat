## 简介

Zhangy Chat R4 是一个深度智能的本地 AI 助手，在 R3 基础上新增**逻辑推理引擎**和**情感计算引擎**，实现从"数据匹配式问答"到"智能逻辑推理"的突破，从"程序化安慰"到"真实情感共情"的升级。

## 核心特性

### 🧠 R4 新增功能

#### 逻辑推理引擎
- **语义解析**: 识别问题类型（为什么/怎么做/要不要/哪个更好）
- **逻辑推理**: 基于因果、流程、决策、比较等逻辑推导答案
- **情境适配**: 结合用户情绪与上下文调整回答角度
- **答案生成**: 输出有逻辑、有温度的回答，而非标准答案

#### 情感计算引擎
- **深度情感识别**: 识别焦虑、沮丧、生气、疲惫、压力、开心、自豪、孤独等情绪
- **情感记忆**: 记住用户近期情绪状态（7 天），实现上下文情感衔接
- **真实共情回复**: 先接纳→再理解→最后轻量建议，不是程序化安慰
- **情感强度调节**: 支持弱/中/强三档，用户可自定义共情深度

#### 智能情境适配
- **负面情绪**: 先共情（换谁都会难过）→ 再理解（你已经很努力了）→ 最后建议
- **正面情绪**: 真诚共鸣（太为你开心了）→ 强化成就感（这是你应得的）
- **中性问题**: 直接解答，适度关怀

## 快速开始

### 环境要求

- Python 3.8+
- 8GB+ RAM
- Windows 10+ / macOS 12+ / Linux (Ubuntu 20.04+)

### 安装

```bash
cd zhangy_chat
pip install -r requirements.txt
```

### 运行

```bash
# GUI 图形界面模式
python gui.py

# CMD 命令行模式
python main.py -cmd
```

## 使用示例

### 逻辑推理问答

```
用户：为什么加班越久效率越低？

AI: 关于「加班效率」，我来帮你分析一下背后的原因：

**可能的原因**:
1. 注意力是有限资源，长时间工作会消耗
2. 大脑疲劳后，单位时间产出下降
3. 熬夜加班影响第二天状态，形成恶性循环

**建议思路**:
- 先明确具体是什么情况
- 找出关键影响因素
- 逐一分析每个因素的作用

你能再多说点具体情境吗？这样我能分析得更准确～
```

```

AI: 加班到现在确实太熬人了，是不是连喝口水的时间都没有？
换谁遇到这种情况都会烦躁的，别憋着。

我知道你已经很努力想把手头的事做好了。
先缓一缓，不用急着做事。

关于你说的这件事，我来帮你分析一下：
...
```

### 情感记忆衔接

```
上午 用户：项目出错被批评了
AI: 被批评确实很难受，别太怪自己...

下午 用户：不想干活
AI: 还在为上午项目的事难过吗？
别太怪自己，出错也是改 bug 的机会呀。
累了就歇歇，不用有负罪感。
```

## 配置说明

### 情感强度等级

| 等级 | 说明 |
|------|------|
| 弱 | 侧重高效解答，最小化情感回应 |
| 中 | 平衡解答与共情（默认） |
| 强 | 侧重情感共情，充分接纳和理解 |

### 逻辑推理问题类型

| 类型 | 疑问词 | 推理方式 |
|------|--------|----------|
| 因果推理 | 为什么/为啥 | 分析原因和结果 |
| 流程推理 | 怎么/如何 | 给出行动框架 |
| 决策推理 | 要不要/该不该 | 梳理决策思路 |
| 比较推理 | 哪个更好/区别 | 对比分析维度 |

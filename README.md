## 📖 简介

**zhangy-chat R3** 是一个具备**深度思考能力**的本地 AI 助手。区别于传统机械式 AI，它能在回应前先思考用户的真实需求、情绪状态和上下文背景，然后给出有针对性、有温度的回答。

### 核心特性

| 特性 | 说明 |
|------|------|
| 🧠 **深度思考** | 多维度分析问题（时间/决策/情绪/资源/人际），适合复杂决策 |
| 💭 **思考式响应** | 先思考再回应，识别需求、情绪，给出个性化回答 |
| 💬 **双界面** | 现代化 GUI（类似微信/豆包）+ CMD 命令行 |
| 🔒 **本地运行** | 所有数据本地存储，不上传云端，隐私安全 |
| 📚 **超全知识库** | 覆盖 12 大领域：饮食、编程、工作、学习、情感、健康等 |
| 🤖 **外部 AI 集成** | 支持 DeepSeek/文心一言/通义千问（可选） |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 8GB+ RAM
- Windows 10+ / macOS 12+ / Linux

### 安装

```bash
cd zhangy_chat
pip install -r requirements.txt
```

### 运行

```bash
# GUI 图形界面模式（推荐）
python gui.py

# CMD 命令行模式
python main.py -cmd
```

## 💡 核心功能

### 1. 🧠 深度思考模式

**适合场景**：复杂决策、情绪问题、长期规划

**使用方式**：
- **GUI**: 勾选"🧠 深度思考"开关
- **CMD**: 使用 `/deep <问题>` 指令

**示例**：
```
/deep 要不要辞职

输出：
[深度思考] 分析维度：决策维度：需要权衡利弊得失 | 情绪维度：需要关注情感需求 | 
推理：第一步：列出所有可选方案 → 第二步：分析每个方案的风险和收益 → 
第三步：评估个人承受能力和优先级 | 建议：这个问题比较复杂，建议分步骤处理

## 工作建议
**职场生存**:
- 少说话多做事，尤其是新人
- 老板画饼听听就好，实际利益最重要
...
```

**思考维度**：
- ⏰ **时间维度**：短期和长期影响
- ⚖️ **决策维度**：权衡利弊得失
- 💚 **情绪维度**：关注情感需求
- 💰 **资源维度**：考虑资源限制
- 👥 **人际维度**：考虑他人感受

### 2. 💬 双界面支持

#### GUI 图形界面（类似微信/豆包）

**特点**：
- 📱 左侧对话列表，右侧聊天窗口
- ➕ 支持多个对话，随时切换
- 🧠 深度思考开关
- 🎨 豆包蓝配色，气泡式对话

**启动**：
```bash
python gui.py
```

#### CMD 命令行界面

**特点**：
- ⌨️ 键盘操作，高效快捷
- 📋 丰富的指令系统
- 🧠 支持 `/deep` 深度思考

**启动**：
```bash
python main.py -cmd
```

**常用指令**：
| 指令 | 说明 | 示例 |
|------|------|------|
| `/add <标题>` | 添加任务 | `/add 学习 Python` |
| `/list` | 查看任务 | `/list pending` |
| `/deep <问题>` | 深度思考 | `/deep 要不要辞职` |
| `/think on/off` | 开启/关闭思考 | `/think on` |
| `/mood [心情]` | 设置心情 | `/mood 高效` |
| `/gui` | 切换图形界面 | `/gui` |
| `/exit` | 退出 | `/exit` |

## ⚙️ 配置说明

### 内存配置

| 内存 | 缓存 | 并发 | 数据加载 |
|------|------|------|----------|
| 8GiB | 64MB | 2 | 分块加载 |
| 16GiB | 256MB | 4 | 分块加载 |
| 32GiB | 512MB | 8 | 混合加载 |
| 64GiB | 1024MB | 16 | 全量加载 |

**设置方式**：
- GUI: 设置窗口 → 内存配置
- CMD: `/mem [8/16/32/64]`

### 心情标签

| 心情 | 回应风格 |
|------|----------|
| 😟 焦虑/低落 | 共情优先，语气温柔 |
| 🎯 高效/专注 | 简洁干练，直奔主题 |
| 😊 轻松/愉悦 | 亲切活泼，轻松互动 |
| 🌿 平静/温和 | 温和平稳，给予安全感 |
| 😴 疲惫/休息 | 关怀为主，强调休息 |

**设置方式**：
- GUI: 心情按钮
- CMD: `/mood [心情]`

## 🔒 隐私保护

- ✅ **本地运行**：所有数据存储在本地 `data/` 目录

- ✅ **备份恢复**：支持手动备份和恢复数据
- ✅ **数据导出**：支持导出 TXT/Excel 格式

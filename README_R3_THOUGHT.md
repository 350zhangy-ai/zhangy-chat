# Zhangy Chat R3 - 思考可见版

## 功能说明

此版本基于 R3 开发，实现了**思考过程可见**功能，类似 MiniMind 的 CoT（Chain of Thought）展示。

## 核心特点

1. **思考过程实时显示**: AI 回答前先显示思考过程
2. **左侧思考区 + 右侧对话区**: 双栏布局，思考过程清晰可见
3. **可开关思考显示**: 用户可选择是否显示思考过程

## 思考过程展示

```
思考中...
  ├─ 情感分析：识别到情绪：平静
  ├─ 匹配实用回复库：找到相关建议
  └─ [OK] 回答生成

---

[正式回答内容]
```

## 运行方式

### GUI 模式（推荐）
```bash
python gui_r3_thought.py
```

界面布局:
- **左侧**: 思考过程显示区（黑底绿字，类似终端效果）
- **右侧**: 对话区 + 输入区
- **底部**: 思考过程开关（可勾选）

### CMD 模式
```bash
python -c "from zhangy_chat.assistant_r3_thought import Assistant; a = Assistant(); print(a.chat('你的问题', show_thought=True))"
```

## 思考过程类型

| 步骤 | 说明 |
|------|------|
| 思考中... | 开始处理问题 |
| 情感分析 | 识别用户情绪状态 |
| 匹配实用回复库 | 查找预设回复 |
| 问题分析 | 未匹配时启动逻辑推理 |
| 推理类型 | 识别问题类型（因果/流程/决策） |
| [OK] 回答生成 | 生成最终回答 |

## 文件说明

| 文件 | 说明 |
|------|------|
| `assistant_r3_thought.py` | 思考可见版 AI 助手核心 |
| `gui_r3_thought.py` | 思考可见版 GUI 界面 |
| `config.yaml` | 配置文件（R3 版本） |

## 与原版 R3 的区别

| 功能 | R3 原版 | R3 思考可见版 |
|------|--------|--------------|
| 思考过程 | 不显示 | 实时显示 |
| 界面布局 | 单栏 | 双栏（思考 + 对话） |
| 响应速度 | 正常 | 略慢（需记录思考） |
| 适用场景 | 日常使用 | 调试/演示 |

## 技术实现

1. **思考记录**: `self.thought_process` 列表记录每一步思考
2. **格式化输出**: `_format_response()` 组装思考过程和回答
3. **GUI 显示**: 左侧文本框实时显示思考过程
4. **可开关**: 通过复选框控制 `show_thought` 参数

## 注意事项

- 此版本为独立版本，不影响原有 R3/R4/R5 代码
- 思考过程会增加少量响应时间
- 适合演示 AI 内部工作原理
- 日常使用可关闭思考显示
